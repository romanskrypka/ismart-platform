# AI-Native Organizations Sprint — Полный комплект учебных материалов

## Обзор курса

**Концепция:** Переход от использования чат-ботов к построению агентной инфраструктуры компании.

**Длительность:** 3 недели (9 основных сессий + 4 кейс-сессии + подготовительный модуль).

**Формат:** Лекция → Coworking → Воркшоп → Кейс от эксперта.

---

## Структура материалов

Для каждой сессии подготовлены три типа документов:

| Тип файла | Описание | Формат |
|-----------|----------|--------|
| `lecture.md` | Лекционный конспект с теорией, примерами и вопросами для обсуждения | Markdown |
| `practice.md` | Практические задания и упражнения с пошаговыми инструкциями | Markdown |
| `templates.md` | Рабочие шаблоны, чек-листы и фреймворки для заполнения | Markdown |
