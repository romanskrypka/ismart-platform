## Шаблон 2: Фреймворк для сравнения моделей «Человек» vs «Агент» vs «Гибрид»

Этот фреймворк поможет вам принять решение, какую модель операционной деятельности выбрать для конкретного бизнес-процесса. Оцените каждый критерий по шкале от 1 до 5, где 1 — минимальное соответствие, 5 — максимальное.

| Бизнес-процесс: [Название процесса] | | | |
| :--- | :--- | :--- | :--- |
| **Критерий оценки** | **Модель «Человек»** | **Модель «Агент»** | **Модель «Гибрид»** |
| | *Оценка (1-5)* | *Оценка (1-5)* | *Оценка (1-5)* |

### Инструкция:

1.  Выберите конкретный бизнес-процесс для анализа (например, «обработка заказов», «онбординг клиентов»).
2.  Для каждой из трех моделей («Человек», «Агент», «Гибрид») оцените по 5-балльной шкале, насколько она соответствует каждому критерию.
3.  Суммируйте баллы для каждой модели.
4.  Модель, набравшая наибольшее количество баллов, является наиболее предпочтительной для данного процесса.

## Чек-лист 3: Оценка влияния AI на штатное расписание

Этот чек-лист поможет вам спрогнозировать, как внедрение AI повлияет на вашу команду, и подготовить план по управлению изменениями.

### Анализ текущих ролей

- [ ] **Составьте список всех ролей**, которые будут затронуты внедрением AI.
- [ ] **Оцените долю рутинных задач** в каждой роли (в %).
- [ ] **Определите, какие задачи могут быть полностью автоматизированы**.
- [ ] **Определите, какие задачи могут быть аугментированы** (усилены) с помощью AI.
- [ ] **Выделите задачи, требующие исключительно человеческого участия** (креативность, стратегия, эмпатия).

### Прогнозирование новых ролей

- [ ] **Определите потребность в новых специалистах**:
    - [ ] Промпт-инженеры / AI-тренеры?
    - [ ] Оркестраторы AI-систем?
    - [ ] Аналитики AI-данных?
    - [ ] Специалисты по этике AI?
- [ ] **Разработайте профили должностей** для новых ролей.
- [ ] **Оцените возможность переобучения** текущих сотрудников на новые роли.

### План по управлению персоналом

- [ ] **Разработайте программу коммуникаций**:
    - [ ] Объясните цели и выгоды внедрения AI для компании и сотрудников.
    - [ ] Снизьте страхи и сопротивление изменениям.
- [ ] **Создайте план переобучения и повышения квалификации (reskilling & upskilling)**.
- [ ] **Определите метрики для оценки успешности адаптации команды** (например, уровень вовлеченности, производительность).
- [ ] **Разработайте новую систему мотивации**, поощряющую использование AI и развитие новых навыков.
